# 🛒 E-Commerce Executive Dashboard

An advanced, two-page Power BI analytics solution designed to track business health, regional growth, and customer loyalty patterns using the SuperStore dataset.

## 📊 Dashboard Pages

### 1. Executive Summary
Focuses on high-level financial KPIs and sales distribution.
* **Revenue & Profit:** Tracks $143.50K in revenue with a healthy 2064% Profit Margin (filtered context).
* **Trends:** Yearly revenue analysis from 2014–2017.
* **Geographic/Category Mix:** Interactive breakdown of performance by East/West/Central/South regions and product categories.

### 2. Customers Analytics
Focuses on buyer behavior and individual performance.
* **Retention Tracking:** Visualizes the ratio of Repeat Customers (99.37%) vs. One-Time Customers.
* **Top Performers:** Detailed table ranking customers by Total Revenue, Profit, and contribution percentage.

## 🚀 Technical Stack
* **Format:** Power BI Project (.pbip)
* **Modeling Language:** TMDL (Tabular Model Definition Language)
* **Data Source:** E-commerce dataset (CSV/Excel)
* **DAX Measures:** Custom logic for Average Order Value, Profit Margin %, and Customer Segmentation.
